## Troubleshooting

### "Module not found" error
- Go to Pydroid menu ‚Üí Pip
- Install missing package

### Voice not working
- Check microphone permission in Android Settings ‚Üí Apps ‚Üí Pydroid 3
- Voice only works on Android (not desktop)

### GUI too small
- In Pydroid settings, increase font size
- Or edit line: `self.root.geometry("800x600")` to larger values

### Ardy not responding
- Check internet connection (for web search)
- Restart Pydroid 3
- Memory file may be corrupted - delete `ardy_true_memory.json` and restart
